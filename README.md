# Amazon Fulfillment Center Analytics Dashboard

This project is an interactive Power BI dashboard developed using an Amazon Fulfillment Center dataset sourced from Kaggle. The primary objective is to analyze warehouse operations, sales performance, customer distribution, and shipment tracking while providing actionable business insights through data visualization.

---

## 📌 Overview

The project began with extensive data preprocessing using **Python** and **Pandas**. Data cleaning activities included:
- Handling missing values
- Removing duplicate records
- Validating data consistency
- Transforming the dataset into a structured format suitable for analysis

---

## 📊 Dashboard Features

Using **Power BI**, I designed a dynamic dashboard featuring:

- ✅ **KPIs** — Total Revenue, Total Orders, Total Products Sold, Total Customers
- 📈 **Year-over-Year Sales Trends**
- 🛍️ **Product-wise Sales Analysis**
- 🌍 **Customer Distribution by Country**
- 🏷️ **Brand Share Analysis**
- 🚚 **Shipment Status Monitoring**

---

## 🔍 Key Highlights

- **KPI-driven sales trend analysis** to identify demand spikes and high order volume periods
- **Interactive slicers** to filter data by date range and delivery status
- **Shipment monitoring gauge** to track pending orders and support operational efficiency

---

## 🎯 Objective

To transform raw warehouse data into meaningful insights that support:
- Inventory management
- Sales optimization
- Strategic decision-making
